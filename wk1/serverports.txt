1) What is the definition of server port?
	
	Server ports are the endpoints of a network connection that is used to exchange information between a 
web server and a web client.

2) What is server port range?
	
	Port numbers range from 0 to 65536.

3) How many bits does number above represent?

	Represents 524288 bits.

4) State 10 common server ports

	HTTP – Port 80
	HTTPS – 443
	FTP – 21
	FTPS / SSH – 22
	POP3 – 110
	POP3 SSL – 995
	IMAP – 143
	IMAP SSL – 993
	SMTP – 25 (Alternate: 26)
	SMTP SSL – 587
